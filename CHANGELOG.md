# Changelog

All notable changes to this repository will be documented in this file.

## [1.0.0] - 2025-04-01

### Added
- Initial repository setup
- GitHub Actions workflow for monthly report generation
- Report generation script (Python)
- Initial data file with federal, provincial, and local programs
- Report templates and data schema documentation
- First monthly report (April 2025)

### Federal Programs Tracked
- IRAP (Industrial Research Assistance Program)
- SR&ED (Scientific Research & Experimental Development)
- CanExport SMEs
- CDAP (Canada Digital Adoption Program)
- SDTC (Sustainable Development Technology Canada)
- BDC Financing

### Provincial Programs Tracked
- EODP (Eastern Ontario Development Program)
- IOF (Invest Ontario Fund)
- Regional Development Program
- Summer Company

### Local Programs Tracked
- Kingston: Property tax incentives, Facade Improvement, Business license waivers, Density bonuses
- Belleville: Community Improvement Plan, Tax Increment Financing, Development Charge Credits
- Brockville: Downtown CIP, Brownfield Redevelopment
- Cornwall: Community Improvement Plan, Development Charge Reduction, Property Tax Assistance
- Peterborough: Community Improvement Plan, Heritage Property Grants, Brownfield Remediation, Fast-Track Permitting
