# Report Template

Use this template structure when creating new monthly reports.

## Header Template

```markdown
# Kingston/Eastern Ontario Government Incentives Report

**Report Date:** [Month Year]  
**Generated:** [YYYY-MM-DD HH:MM UTC]  
**Coverage:** Federal, Provincial, and Local (Kingston, Belleville, Brockville, Cornwall, Peterborough)

---

## Executive Summary

[Brief overview of changes this month, new programs, deadlines, etc.]

### Quick Stats
- **Federal Programs Tracked:** X
- **Provincial Programs Tracked:** X
- **Local Jurisdictions:** 5
- **Incentive Types:** 7

---
```

## Section Templates

### Federal Programs Section
Track programs from:
- NRC-IRAP
- CRA (SR&ED)
- Global Affairs Canada (CanExport)
- SDTC
- BDC
- EDC

### Provincial Programs Section
Track programs from:
- Invest Ontario
- FedDev Ontario (EODP)
- Ministry of Economic Development
- Sector-specific programs

### Local Programs Section
Track programs for each municipality:
- Kingston
- Belleville
- Brockville
- Cornwall
- Peterborough
